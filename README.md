# 🚀 跨境运营灵动工作台 (Cross-Border Ops Dashboard) v5.2

一个专为跨境电商运营人员设计的纯前端、离线化、多功能集成工作台。集成了利润核算、物流追踪、任务管理及广告监控等核心功能。

[👉 点击这里立即使用在线版](https://JIILuke-cmd.github.io/Yunyi-2/)

---

## ✨ 核心功能 (v5.2)

### 1. 📋 任务与日程看板
- **重点任务**：核心事项备忘，支持勾选完成。
- **今日日程**：时间轴安排。
- **日常固定**：每日 0 点自动重置状态，适合每日例行检查（如查销量、回邮件）。

### 2. 🎧 售后处理中心
- **智能排序**：支持按“登记日期”或“已过天数”快速筛选积压工单。
- **可视化状态**：清晰展示处理进度，支持大文本记录异常详情。
- **Excel 导出**：一键导出 CSV 表格，方便进行周报/月报汇总。

### 3. 💰 利润即时分析
- **实时汇率**：自动获取最新美元/人民币汇率（支持离线模式）。
- **智能竞价**：输入竞品价格与我司价格，自动判断价格优势区间。
- **精准核算**：自动预扣 4% 平台杂费，计算$35免邮门槛，反推30%利润建议价。

### 4. ✈️ SKU 物流雷达
- **到仓监控**：输入发货日期与预计时效，自动计算预计到仓日。
- **逾期预警**：一旦超过预计日期，状态自动变红闪烁报警。

### 5. 📊 FB 广告表现监控
- **本地化隐私**：图片通过本地上传（Base64），不依赖网络图床，数据存储在本地浏览器。
- **多维数据**：记录广告编码、名称及点赞/评论/心情数据。

---

## 🛠️ 技术特性
- **数据安全**：所有数据存储在浏览器本地 (LocalStorage)，不经过任何服务器，刷新不丢失。
- **纯静态**：无后端，HTML 单文件即可运行。
- **数据迁移**：支持 JSON 格式的一键备份与恢复，方便在公司/家里电脑切换数据。
- **响应式设计**：完美适配 PC 宽屏操作。
